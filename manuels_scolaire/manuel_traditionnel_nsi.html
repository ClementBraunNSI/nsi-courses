Sciences Numériques et Technologie

La spécialité Sciences Numériques et Technologie (SNT) en classe de seconde vise à faire découvrir aux élèves les concepts fondamentaux de l'informatique et du numérique. Cette partie du manuel couvre les thèmes essentiels du programme officiel.

Objectifs de la formation

À l'issue de cette formation, l'élève sera capable de :

  • Comprendre les enjeux du numérique dans la société
  • Maîtriser les concepts de base d'Internet et du Web
  • Découvrir les principes de la programmation
  • Analyser des données numériques simples
  • Développer un esprit critique face aux technologies
Organisation des chapitres

Cette partie est organisée en trois chapitres progressifs qui permettent une découverte structurée du domaine informatique.

Chapitre 1 Internet et le Web

Internet et le Web sont devenus incontournables dans notre société. Ce chapitre explore leur fonctionnement, leur histoire et leurs enjeux.

1.1 Qu'est-ce qu'Internet ?
Définition - Internet

Internet est un réseau mondial d'ordinateurs interconnectés qui communiquent grâce à des protocoles standardisés. C'est l'infrastructure physique et logique qui permet l'échange d'informations à l'échelle planétaire.

Histoire d'Internet

Internet trouve ses origines dans le projet ARPANET, développé par l'armée américaine dans les années 1960. L'objectif était de créer un réseau de communication résistant aux pannes.

Les étapes clés :

  • 1969 : Premier message envoyé sur ARPANET
  • 1983 : Adoption du protocole TCP/IP
  • 1991 : Naissance du World Wide Web
  • 1995 : Commercialisation d'Internet
Architecture d'Internet

Internet repose sur une architecture décentralisée composée de :

  • Fournisseurs d'accès Internet (FAI) : Ils connectent les utilisateurs au réseau
  • Serveurs : Ordinateurs qui hébergent les services et contenus
  • Routeurs : Équipements qui dirigent les données
  • Protocoles : Règles de communication (TCP/IP, HTTP, etc.)

1.2 Le World Wide Web
Définition - World Wide Web

Le Web est un service qui fonctionne sur Internet. Il permet de consulter des pages web reliées entre elles par des liens hypertextes. Il a été inventé par Tim Berners-Lee au CERN en 1991.

Composants du Web

Le Web repose sur trois technologies fondamentales :

Technologie Rôle Exemple
URL Adresse unique d'une ressource https://www.education.gouv.fr
HTTP Protocole de communication Requête GET, POST
HTML Langage de description des pages <h1>Titre</h1>
Fonctionnement d'une requête Web

Lorsque vous tapez une adresse dans votre navigateur :

  1. Le navigateur traduit l'URL en adresse IP (DNS)
  2. Une connexion s'établit avec le serveur web
  3. Le serveur envoie la page HTML demandée
  4. Le navigateur interprète et affiche la page
# Exemple de requête HTTP GET /index.html HTTP/1.1 Host: www.exemple.fr User-Agent: Mozilla/5.0 Accept: text/html

Chapitre 2 Données et Algorithmes

Les données sont au cœur du numérique. Ce chapitre explore comment les représenter, les traiter et les analyser grâce aux algorithmes.

2.1 Représentation des données
Définition - Donnée numérique

Une donnée numérique est une information codée sous forme binaire (0 et 1) que l'ordinateur peut traiter. Elle peut représenter du texte, des nombres, des images, du son, etc.

Types de données
Type Description Exemples
Texte Chaînes de caractères "Bonjour", "NSI"
Nombres Entiers ou décimaux 42, 3.14, -7
Booléens Vrai ou faux True, False
Listes Collections ordonnées [1, 2, 3], ["a", "b"]
Codage des caractères

Les caractères sont codés en nombres. Les standards principaux sont :

  • ASCII : 128 caractères (anglais)
  • UTF-8 : Standard universel (toutes les langues)
# Exemple de codage ASCII A = 65 a = 97 0 = 48 espace = 32

2.2 Introduction aux algorithmes
Définition - Algorithme

Un algorithme est une suite finie d'instructions précises et non ambiguës permettant de résoudre un problème ou d'effectuer une tâche.

Caractéristiques d'un bon algorithme
  • Précision : Chaque étape est clairement définie
  • Finitude : L'algorithme se termine en un temps fini
  • Efficacité : Il résout le problème rapidement
  • Généralité : Il fonctionne pour tous les cas du problème
Exemple : Algorithme de recherche

Problème : Trouver un mot dans un dictionnaire

Algorithme de recherche dichotomique : 1. Ouvrir le dictionnaire au milieu 2. Si le mot cherché est sur cette page : TROUVÉ 3. Si le mot est avant : chercher dans la première moitié 4. Si le mot est après : chercher dans la seconde moitié 5. Répéter jusqu'à trouver le mot
Structures algorithmiques de base
Structure Description Exemple
Séquence Instructions successives Étape 1, puis étape 2, puis étape 3
Condition Choix selon un test Si il pleut, prendre un parapluie
Répétition Boucle d'instructions Tant que pas arrivé, continuer

Chapitre 3 Programmation Python

Python est un langage de programmation simple et puissant, idéal pour débuter. Ce chapitre présente les bases de la programmation avec Python.

3.1 Premiers pas avec Python
Qu'est-ce que Python ?

Python est un langage de programmation créé en 1991 par Guido van Rossum. Il est réputé pour sa simplicité et sa lisibilité, ce qui en fait un excellent choix pour l'apprentissage.

Premier programme

Traditionnellement, le premier programme affiche "Hello World" :

print("Hello World") print("Bonjour NSI !")

La fonction print() permet d'afficher du texte à l'écran.

Variables et affectation

Une variable est un nom qui désigne une valeur en mémoire :

# Déclaration de variables nom = "Alice" age = 16 taille = 1.65 eleve = True # Affichage print("Nom :", nom) print("Age :", age, "ans")
Types de données en Python
Type Python Description Exemple
str Chaîne de caractères "Bonjour"
int Nombre entier 42
float Nombre décimal 3.14
bool Booléen True, False
list Liste [1, 2, 3]

Glossaire des termes informatiques
Algorithme

Suite finie d'instructions précises permettant de résoudre un problème.

ASCII

American Standard Code for Information Interchange. Standard de codage des caractères sur 7 bits (128 caractères).

Bit

Unité élémentaire d'information en informatique, pouvant prendre la valeur 0 ou 1.

HTML

HyperText Markup Language. Langage de balisage pour créer des pages web.

HTTP

HyperText Transfer Protocol. Protocole de communication du Web.

Internet

Réseau mondial d'ordinateurs interconnectés utilisant le protocole TCP/IP.

Python

Langage de programmation interprété, orienté objet et de haut niveau.

URL

Uniform Resource Locator. Adresse unique d'une ressource sur Internet.

Variable

Nom symbolique associé à une valeur qui peut changer au cours de l'exécution d'un programme.

Web

World Wide Web. Service fonctionnant sur Internet permettant de consulter des pages reliées par des liens hypertextes.
