Internet

Sciences Numériques et Technologie - Seconde

Présentation du cours

Ce cours vous permettra de comprendre les fondements d'Internet, son histoire, son fonctionnement et les technologies qui le composent. Vous découvrirez comment les machines communiquent entre elles à l'échelle mondiale.

Point historique

La création d'Internet date du début des années 1960. L'idée était de relier divers machines pour envoyer des données, à l'origine des travaux issus de laboratoires ou des universités.

A l'époque, le projet initial s'appelait ARPANET et était un projet détenu par la Défense des États-Unis chapeauté Robert Kahn à la Defense Advanced Research Projects Agency. Le premier objectif était de relier les universités de Stanford, de Los Angeles et celle de l'Utah.

Ce n'est que le 20 septembre 1969 que la première communication voit le jour entre l'université de Californie et celle de Stanford. Le premier message transmis de l'Histoire était login.

À la suite de cela, dans les années 70-80, des normalismes de communication ont vu le jour. On appelle cet ensemble de règles à respecter des protocoles. La manière de communiquer pour les machines est découpées en diverses étapes. Ces diverses étapes sont catégorisées dans un modèle : le modèle TCP-IP qui explique chaque étape de la communication.

Dans les années 1980, le fameux protocole TCP/IPv4 a vu le jour. Il est installé en 1983 sur ARPANET, la même année où ont vu le jour les règles des systèmes de nom de domaine (DNS).

Les années 1990 ont permis de faire voir le jour à une des plus grandes technologies jamais créée par l'être humain : le WEB, créé au CERN (en Suisse) par Tim Berners-Lee et Robert Cailliau.

Depuis, Internet permet de relier plus de 3 à 4 milliards d'internautes pour s'envoyer des mails, des fichiers ou accéder à une quantité de données incommensurable à l'aide de plusieurs applications dépendant de protocoles. Par exemple, il existe des applications de mail (protocole POP), d'échange de fichier (FTP) ou de navigation sur des pages (WEB).

En clair, Internet est un réseau informatique à échelle mondiale sur lequel de nombreuses applications sont basées.

Notion de réseaux

Un réseau informatique est un ensemble d'équipements reliés entre eux pour échanger des informations. Parmi ces équipements, on retrouve :

Élément Description
Ordinateurs Machines qui exécutent des programmes informatiques
Serveurs Ordinateurs qui fournissent des ressources aux autres ordinateurs du réseau
Routeurs Équipements qui permettent de relier plusieurs réseaux entre eux
Commutateurs (switch) Équipements qui relient plusieurs ordinateurs au sein d'un même réseau

Internet est un réseau mondial d'ordinateurs reliés entre eux. Il permet d'échanger des informations à l'échelle mondiale.

Protocoles

Pour que les équipements puissent communiquer entre eux sur Internet, ils utilisent des protocoles. Un protocole est un ensemble de règles qui définissent comment les données sont échangées.

Protocole TCP/IP

Le protocole TCP/IP (Transmission Control Protocol/Internet Protocol) est le protocole de base d'Internet. Il permet de découper les données en paquets, de les acheminer vers leur destination et de les réassembler.

Protocole HTTP/HTTPS

Le protocole HTTP (HyperText Transfer Protocol) permet d'échanger des documents hypertextes (pages web) sur Internet. HTTPS est la version sécurisée de HTTP, qui chiffre les données échangées.

Protocole DNS

Le protocole DNS (Domain Name System) permet de traduire les noms de domaine (comme www.example.com) en adresses IP (comme 192.168.1.1).

Adressages

Pour communiquer sur un réseau, chaque machine doit posséder une adresse unique. Il existe deux types d'adresses :

Adresse MAC

L'adresse MAC (Media Access Control) est une adresse physique attribuée à la carte réseau d'un équipement. Elle est unique et ne change pas.

Format : 6 octets en hexadécimal, séparés par des tirets ou des deux-points.
Exemple : 00:1A:2B:3C:4D:5E

Adresse IP

L'adresse IP est une adresse logique attribuée à un équipement sur un réseau. Elle peut changer selon le réseau auquel l'équipement est connecté.

Format IPv4 : 4 octets en décimal, séparés par des points.
Exemple : 192.168.1.1

Format IPv6 : 8 groupes de 4 chiffres hexadécimaux, séparés par des deux-points.
Exemple : 2001:0db8:85a3:0000:0000:8a2e:0370:7334

La communication entre deux machines

Pour que deux machines puissent communiquer sur Internet, elles utilisent un système d'adressage et des protocoles de communication.

Étapes de communication

  1. La machine source découpe les données en paquets.
  2. Chaque paquet est envoyé sur le réseau avec l'adresse IP de destination.
  3. Les routeurs acheminent les paquets vers leur destination.
  4. La machine de destination réassemble les paquets pour reconstituer les données d'origine.

Ce système permet d'échanger des données entre n'importe quelles machines connectées à Internet, peu importe leur emplacement géographique.
