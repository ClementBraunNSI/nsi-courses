Cours 2 - Types de données en Python

Numérique et Sciences Informatiques - Première

Les types de données en Python

En programmation, les données manipulées sont classées par types. Le type d'une donnée détermine :

  • Les valeurs possibles pour cette donnée
  • Les opérations qu'on peut effectuer sur cette donnée
  • La façon dont cette donnée est stockée en mémoire

Python est un langage à typage dynamique, ce qui signifie que le type d'une variable est déterminé automatiquement lors de l'exécution du programme, et peut changer au cours de l'exécution.

Les types de base

Les booléens (bool)

Le type booléen ne peut prendre que deux valeurs : True (vrai) ou False (faux). Il est utilisé pour représenter des conditions logiques.

# Déclaration de variables booléennes
a = True
b = False

# Opérations logiques
print(a and b)  # ET logique : False
print(a or b)   # OU logique : True
print(not a)    # NON logique : False

# Comparaisons qui renvoient des booléens
print(5 > 3)    # True
print(5 == 3)   # False
print(5 != 3)   # True

Les entiers (int)

Le type entier représente des nombres entiers (positifs, négatifs ou nul) sans partie décimale.

# Déclaration de variables entières
a = 42
b = -7
c = 0

# Opérations arithmétiques
print(a + b)    # Addition : 35
print(a - b)    # Soustraction : 49
print(a * b)    # Multiplication : -294
print(a // b)   # Division entière : -6
print(a % b)    # Modulo (reste de la division) : 0
print(a ** 2)   # Puissance : 1764

Les flottants (float)

Le type flottant représente des nombres à virgule flottante, c'est-à-dire des nombres avec une partie décimale.

# Déclaration de variables flottantes
a = 3.14
b = -2.5
c = 1.0

# Opérations arithmétiques
print(a + b)    # Addition : 0.64
print(a - b)    # Soustraction : 5.64
print(a * b)    # Multiplication : -7.85
print(a / b)    # Division : -1.256
print(a ** 2)   # Puissance : 9.8596

# Attention aux erreurs d'arrondi
print(0.1 + 0.2)  # 0.30000000000000004

Les chaînes de caractères (str)

Le type chaîne de caractères représente une séquence de caractères. En Python, les chaînes sont délimitées par des guillemets simples ou doubles.

# Déclaration de chaînes de caractères
prenom = "Alice"
nom = 'Dupont'

# Concaténation
nom_complet = prenom + " " + nom
print(nom_complet)  # Alice Dupont

# Répétition
print(prenom * 3)  # AliceAliceAlice

# Accès aux caractères (indexation)
print(prenom[0])  # A (le premier caractère)
print(prenom[-1])  # e (le dernier caractère)

# Sous-chaînes (slicing)
print(prenom[1:3])  # li (caractères d'index 1 à 2)

# Longueur d'une chaîne
print(len(prenom))  # 5

# Méthodes sur les chaînes
print(prenom.upper())  # ALICE
print(prenom.lower())  # alice
print("  texte  ".strip())  # "texte" (supprime les espaces)
print("bonjour monde".replace("monde", "Python"))  # bonjour Python

Conversion entre types

Python permet de convertir des données d'un type à un autre à l'aide de fonctions de conversion.

# Conversion en entier
print(int(3.14))  # 3 (troncature)
print(int("42"))  # 42
print(int(True))  # 1
print(int(False))  # 0

# Conversion en flottant
print(float(42))  # 42.0
print(float("3.14"))  # 3.14
print(float(True))  # 1.0

# Conversion en booléen
print(bool(1))  # True
print(bool(0))  # False
print(bool("bonjour"))  # True (chaîne non vide)
print(bool(""))  # False (chaîne vide)

# Conversion en chaîne
print(str(42))  # "42"
print(str(3.14))  # "3.14"
print(str(True))  # "True"

Attention aux erreurs de conversion

Toutes les conversions ne sont pas possibles. Par exemple :

# Ces conversions génèrent des erreurs
int("bonjour")  # ValueError: invalid literal for int()
float("texte")  # ValueError: could not convert string to float

Vérification de type

Pour connaître le type d'une variable, on peut utiliser la fonction type(). Pour vérifier si une variable est d'un type spécifique, on peut utiliser la fonction isinstance().

# Utilisation de type()
a = 42
print(type(a))  # 

b = 3.14
print(type(b))  # 

c = "bonjour"
print(type(c))  # 

d = True
print(type(d))  # 

# Utilisation de isinstance()
print(isinstance(a, int))  # True
print(isinstance(b, float))  # True
print(isinstance(c, str))  # True
print(isinstance(d, bool))  # True
print(isinstance(a, float))  # False

Exercices pratiques

Exercice 1 : Manipulation de types

Écrivez un programme qui demande à l'utilisateur de saisir un nombre, puis affiche :

  • Le type de la donnée saisie
  • La conversion en entier (si possible)
  • La conversion en flottant
  • La conversion en booléen
# Programme de manipulation de types
saisie = input("Entrez un nombre : ")

print(f"Type de la saisie : {type(saisie)}")

try:
    entier = int(saisie)
    print(f"Conversion en entier : {entier} (type: {type(entier)})")
except ValueError:
    print("Impossible de convertir en entier")

try:
    flottant = float(saisie)
    print(f"Conversion en flottant : {flottant} (type: {type(flottant)})")
except ValueError:
    print("Impossible de convertir en flottant")

boolean = bool(saisie)
print(f"Conversion en booléen : {boolean} (type: {type(boolean)})")

Exercice 2 : Calcul d'IMC

Écrivez un programme qui calcule l'Indice de Masse Corporelle (IMC) d'une personne. Le programme doit :

  • Demander le poids en kg (nombre à virgule)
  • Demander la taille en m (nombre à virgule)
  • Calculer l'IMC selon la formule : IMC = poids / (taille²)
  • Afficher l'IMC avec 2 décimales
# Programme de calcul d'IMC
poids = float(input("Entrez votre poids en kg : "))
taille = float(input("Entrez votre taille en m : "))

imc = poids / (taille ** 2)

print(f"Votre IMC est de {imc:.2f}")

# Interprétation de l'IMC
if imc < 18.5:
    print("Vous êtes en sous-poids")
elif imc < 25:
    print("Votre poids est normal")
elif imc < 30:
    print("Vous êtes en surpoids")
else:
    print("Vous êtes en obésité")
