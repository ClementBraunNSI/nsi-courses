Cours 1 - Constructions élémentaires en Python

Numérique et Sciences Informatiques - Première

Qu'est-ce qu'un programme ?

Un programme est une suite d'instructions élémentaires destinées à être exécutées par un ordinateur. Le langage Python est un langage dit de haut niveau, ce qui signifie qu'il est plus proche de nous (utilisateurs) que de l'ordinateur (du processeur).

Ce langage est basé sur nos ordinateurs qui réalisent des instructions de manière séquentielle (c'est-à-dire réalisées les unes après les autres). On peut dénombrer un certain nombre d'instructions élémentaires qui permettent de créer un programme.

Les instructions élémentaires

L'instantiation

L'instantiation consiste à associer une valeur à une variable. En Python, on utilise l'opérateur = pour cela.

a = 5  # La variable a prend la valeur 5
b = "Bonjour"  # La variable b prend la valeur "Bonjour"

Les opérations

Python permet d'effectuer diverses opérations sur les variables :

Opérations mathématiques

# Addition
a = 5 + 3  # a vaut 8

# Soustraction
b = 10 - 4  # b vaut 6

# Multiplication
c = 3 * 4  # c vaut 12

# Division
d = 10 / 2  # d vaut 5.0 (un float)

# Division entière
e = 10 // 3  # e vaut 3

# Modulo (reste de la division)
f = 10 % 3  # f vaut 1

# Puissance
g = 2 ** 3  # g vaut 8

Opérations de comparaison

# Égalité
a == b  # Renvoie True si a est égal à b, False sinon

# Différence
a != b  # Renvoie True si a est différent de b, False sinon

# Supérieur
a > b  # Renvoie True si a est supérieur à b, False sinon

# Inférieur
a < b  # Renvoie True si a est inférieur à b, False sinon

# Supérieur ou égal
a >= b  # Renvoie True si a est supérieur ou égal à b, False sinon

# Inférieur ou égal
a <= b  # Renvoie True si a est inférieur ou égal à b, False sinon

Les types de données

En Python, chaque variable a un type qui détermine les opérations qu'on peut effectuer dessus.

# Pour connaître le type d'une variable
a = 5
type(a)  # Renvoie <class 'int'>

b = "Bonjour"
type(b)  # Renvoie <class 'str'>

c = 3.14
type(c)  # Renvoie <class 'float'>

d = True
type(d)  # Renvoie <class 'bool'>

Les types de base en Python sont :

  • int : nombres entiers
  • float : nombres à virgule flottante
  • str : chaînes de caractères
  • bool : booléens (True ou False)

Entrées/Sorties

Afficher des informations

La fonction print() permet d'afficher des informations à l'écran.

print("Bonjour le monde !")  # Affiche: Bonjour le monde !

a = 5
print(a)  # Affiche: 5

# On peut combiner plusieurs éléments
print("La valeur de a est", a)  # Affiche: La valeur de a est 5

# Formatage de chaînes
nom = "Alice"
age = 20
print(f"{nom} a {age} ans")  # Affiche: Alice a 20 ans

Récupérer des entrées utilisateur

La fonction input() permet de demander une information à l'utilisateur.

# Demande le nom de l'utilisateur
nom = input("Comment vous appelez-vous ? ")
print(f"Bonjour {nom} !")

# Attention : input() renvoie toujours une chaîne de caractères
age_str = input("Quel âge avez-vous ? ")
age = int(age_str)  # Conversion en entier
print(f"Dans 10 ans, vous aurez {age + 10} ans")

# Version plus concise
age = int(input("Quel âge avez-vous ? "))
print(f"Dans 10 ans, vous aurez {age + 10} ans")

Exercices pratiques

Exercice 1 : Calcul d'aire

Écrivez un programme qui demande à l'utilisateur la longueur et la largeur d'un rectangle, puis affiche son aire.

# Programme de calcul d'aire d'un rectangle
longueur = float(input("Entrez la longueur du rectangle : "))
largeur = float(input("Entrez la largeur du rectangle : "))

aire = longueur * largeur

print(f"L'aire du rectangle est de {aire} unités carrées")

Exercice 2 : Conversion de température

Écrivez un programme qui convertit une température de Celsius en Fahrenheit. La formule est : F = (C × 9/5) + 32

# Programme de conversion de température
celsius = float(input("Entrez une température en degrés Celsius : "))

fahrenheit = (celsius * 9/5) + 32

print(f"{celsius}°C équivaut à {fahrenheit}°F")
