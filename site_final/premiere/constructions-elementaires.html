Constructions élémentaires

Numérique et Sciences Informatiques - Première

Présentation du cours

Ce chapitre vous permettra de comprendre les fondements de la programmation en Python. Vous découvrirez les constructions élémentaires qui sont à la base de tout programme informatique : variables, opérations, conditions, boucles et fonctions.

Qu'est-ce qu'un programme ?

Un programme est une suite d'instructions élémentaires destinées à être exécutées par un ordinateur. Le langage Python est un langage dit de haut niveau, ce qui signifie qu'il est plus proche de nous (utilisateurs) que de l'ordinateur (du processeur).

Ce langage est basé sur nos ordinateurs qui réalisent des instructions de manière séquentielle (c'est-à-dire réalisées les unes après les autres). On peut dénombrer un certain nombre d'instructions élémentaires qui permettent de créer un programme.

Les types de données en Python

Python utilise différents types de données pour représenter les informations :

Les booléens

Le type booléen en Python est utilisé pour représenter des valeurs de vérité. Il n'a que deux valeurs possibles : True (vrai) et False (faux). Les booléens sont souvent utilisés dans les conditions et les boucles pour contrôler le flux d'exécution d'un programme.

Les nombres

Python gère plusieurs types de nombres :

  • Entiers (int) : représentent des nombres entiers comme 42, -10, 0
  • Flottants (float) : représentent des nombres à virgule comme 3.14, -0.5, 2.0

Les chaînes de caractères

Les chaînes de caractères (str) permettent de représenter du texte. Elles sont délimitées par des guillemets simples ou doubles.

message = "Bonjour le monde !"

Structures de contrôle

Les conditions

Les structures conditionnelles permettent d'exécuter différentes portions de code selon qu'une condition est vraie ou fausse.

age = 18
if age >= 18:
    print("Vous êtes majeur")
else:
    print("Vous êtes mineur")

Les boucles

Les boucles permettent de répéter l'exécution d'un bloc de code plusieurs fois.

Boucle for : utilisée pour itérer sur une séquence (comme une liste, un tuple, un dictionnaire, un ensemble ou une chaîne).

for i in range(5):
    print(i)  # Affiche 0, 1, 2, 3, 4

Boucle while : exécute un bloc de code tant qu'une condition est vraie.

compteur = 0
while compteur < 5:
    print(compteur)
    compteur += 1  # Affiche 0, 1, 2, 3, 4

Les fonctions

Les fonctions sont des blocs de code réutilisables qui effectuent une tâche spécifique. Elles permettent de structurer le code et d'éviter les répétitions.

def aire_rectangle(longueur, largeur):
    """Calcule l'aire d'un rectangle"""
    return longueur * largeur

# Appel de la fonction
resultat = aire_rectangle(5, 3)
print(f"L'aire du rectangle est {resultat}")

Les fonctions peuvent avoir des paramètres (entrées) et retourner des valeurs (sorties). Elles peuvent également inclure une documentation (docstring) qui explique leur fonctionnement.
