Exercices sur les boucles

Numérique et Sciences Informatiques - Première

Objectifs des exercices

Cette fiche d'exercices vous permettra de pratiquer l'utilisation des boucles en Python. Vous apprendrez à utiliser les boucles for et while pour résoudre différents problèmes algorithmiques.

Les exercices sont classés par niveau de difficulté croissant. N'hésitez pas à consulter les cours sur les structures de contrôle pour vous aider.

Exercice 1 : Boucles for avec range()

1.1 Affichage des nombres

Écrivez un programme qui utilise une boucle for pour afficher tous les nombres de 1 à 10.

for i in range(1, 11):
    print(i)

1.2 Somme des nombres

Écrivez un programme qui calcule la somme des nombres de 1 à 100 en utilisant une boucle for.

somme = 0
for i in range(1, 101):
    somme += i
print(f"La somme des nombres de 1 à 100 est {somme}")

1.3 Nombres pairs

Écrivez un programme qui affiche tous les nombres pairs entre 1 et 20 en utilisant une boucle for.

# Méthode 1 : en utilisant range avec un pas de 2
for i in range(2, 21, 2):
    print(i)

# Méthode 2 : en utilisant une condition
for i in range(1, 21):
    if i % 2 == 0:
        print(i)

Exercice 2 : Boucles for pour parcourir des séquences

2.1 Parcours d'une chaîne de caractères

Écrivez un programme qui compte le nombre de voyelles dans une chaîne de caractères.

def compter_voyelles(texte):
    voyelles = "aeiouyAEIOUY"
    compteur = 0
    for caractere in texte:
        if caractere in voyelles:
            compteur += 1
    return compteur

# Test de la fonction
texte = "Bonjour tout le monde"
nombre_voyelles = compter_voyelles(texte)
print(f"Le texte '{texte}' contient {nombre_voyelles} voyelles.")

2.2 Parcours d'une liste

Écrivez un programme qui calcule la moyenne des éléments d'une liste de nombres.

def calculer_moyenne(nombres):
    if len(nombres) == 0:
        return 0
    somme = 0
    for nombre in nombres:
        somme += nombre
    return somme / len(nombres)

# Test de la fonction
notes = [12, 15, 18, 10, 14]
moyenne = calculer_moyenne(notes)
print(f"La moyenne des notes est {moyenne}")

2.3 Parcours avec énumération

Écrivez un programme qui utilise enumerate() pour afficher l'index et la valeur de chaque élément d'une liste.

fruits = ["pomme", "banane", "orange", "kiwi", "fraise"]

for index, fruit in enumerate(fruits):
    print(f"Index {index}: {fruit}")

Exercice 3 : Boucles while

3.1 Compte à rebours

Écrivez un programme qui utilise une boucle while pour afficher un compte à rebours de 10 à 1, puis affiche "Décollage !".

compteur = 10
while compteur > 0:
    print(compteur)
    compteur -= 1
print("Décollage !")

3.2 Devinette

Écrivez un programme de devinette où l'utilisateur doit deviner un nombre entre 1 et 100. Le programme indique si le nombre proposé est trop grand ou trop petit, jusqu'à ce que l'utilisateur trouve le bon nombre.

import random

def jeu_devinette():
    nombre_secret = random.randint(1, 100)
    essais = 0
    trouve = False
    
    print("J'ai choisi un nombre entre 1 et 100. À vous de le deviner !")
    
    while not trouve:
        try:
            proposition = int(input("Votre proposition : "))
            essais += 1
            
            if proposition < nombre_secret:
                print("C'est plus !")
            elif proposition > nombre_secret:
                print("C'est moins !")
            else:
                trouve = True
                print(f"Bravo ! Vous avez trouvé en {essais} essais.")
        except ValueError:
            print("Veuillez entrer un nombre valide.")

# Lancer le jeu
jeu_devinette()

3.3 Calcul de la racine carrée

Écrivez un programme qui calcule la racine carrée d'un nombre en utilisant la méthode de Newton (ou méthode de la tangente).

def racine_carree(n, precision=0.0001):
    """Calcule la racine carrée d'un nombre en utilisant la méthode de Newton.
    
    Args:
        n (float): Le nombre dont on veut calculer la racine carrée.
        precision (float, optional): La précision souhaitée. Par défaut 0.0001.
        
    Returns:
        float: La racine carrée approximative de n.
    """
    if n < 0:
        raise ValueError("Impossible de calculer la racine carrée d'un nombre négatif")
    if n == 0:
        return 0
    
    # Valeur initiale
    x = n / 2
    
    # Itérations de la méthode de Newton
    while True:
        # Calcul de la prochaine approximation
        prochaine_x = 0.5 * (x + n / x)
        
        # Vérifier si la précision souhaitée est atteinte
        if abs(prochaine_x - x) < precision:
            return prochaine_x
        
        x = prochaine_x

# Test de la fonction
nombre = 16
racine = racine_carree(nombre)
print(f"La racine carrée de {nombre} est approximativement {racine}")

Exercice 4 : Boucles imbriquées

4.1 Table de multiplication

Écrivez un programme qui affiche la table de multiplication de 1 à 10 en utilisant des boucles imbriquées.

print("Table de multiplication de 1 à 10 :")

for i in range(1, 11):
    for j in range(1, 11):
        print(f"{i} × {j} = {i * j}")
    print("-" * 20)  # Séparateur entre les tables

4.2 Motifs d'étoiles

Écrivez un programme qui affiche un triangle d'étoiles comme celui-ci :

*
**
***
****
*****
hauteur = 5

for i in range(1, hauteur + 1):
    print("*" * i)

4.3 Motif pyramide

Écrivez un programme qui affiche une pyramide d'étoiles comme celle-ci :

    *
   ***
  *****
 *******
*********
hauteur = 5

for i in range(hauteur):
    espaces = " " * (hauteur - i - 1)
    etoiles = "*" * (2 * i + 1)
    print(espaces + etoiles)

Exercice 5 : Projet - Jeu du pendu

Créez un programme qui implémente le jeu du pendu en utilisant des boucles.

import random

def jeu_pendu():
    # Liste de mots possibles
    mots = ["python", "programmation", "informatique", "algorithme", "variable", "fonction", "boucle", "condition"]
    
    # Choisir un mot au hasard
    mot_secret = random.choice(mots)
    lettres_trouvees = ["_"] * len(mot_secret)
    lettres_essayees = []
    essais_restants = 7
    
    print("Bienvenue dans le jeu du Pendu !")
    print(f"Le mot à deviner contient {len(mot_secret)} lettres.")
    
    # Boucle principale du jeu
    while essais_restants > 0 and "_" in lettres_trouvees:
        # Afficher l'état actuel
        print("\nMot : ", " ".join(lettres_trouvees))
        print(f"Essais restants : {essais_restants}")
        print(f"Lettres déjà essayées : {', '.join(lettres_essayees)}")
        
        # Demander une lettre
        lettre = input("Proposez une lettre : ").lower()
        
        # Vérifier si la lettre est valide
        if len(lettre) != 1 or not lettre.isalpha():
            print("Veuillez entrer une seule lettre.")
            continue
        
        # Vérifier si la lettre a déjà été essayée
        if lettre in lettres_essayees:
            print("Vous avez déjà essayé cette lettre.")
            continue
        
        # Ajouter la lettre aux lettres essayées
        lettres_essayees.append(lettre)
        
        # Vérifier si la lettre est dans le mot
        if lettre in mot_secret:
            print("Bonne lettre !")
            # Mettre à jour les lettres trouvées
            for i in range(len(mot_secret)):
                if mot_secret[i] == lettre:
                    lettres_trouvees[i] = lettre
        else:
            print("Cette lettre n'est pas dans le mot.")
            essais_restants -= 1
    
    # Fin du jeu
    if "_" not in lettres_trouvees:
        print(f"\nBravo ! Vous avez trouvé le mot : {mot_secret}")
    else:
        print(f"\nDommage ! Le mot était : {mot_secret}")

# Lancer le jeu
if __name__ == "__main__":
    jeu_pendu()
