Cours 3 - Structures de contrôle en Python

Numérique et Sciences Informatiques - Première

Les structures de contrôle

Les structures de contrôle permettent de modifier le flux d'exécution d'un programme. Elles sont essentielles pour créer des programmes qui peuvent prendre des décisions et répéter des actions. En Python, il existe deux types principaux de structures de contrôle :

  • Les structures conditionnelles : permettent d'exécuter du code en fonction de conditions
  • Les structures itératives (boucles) : permettent de répéter l'exécution d'un bloc de code

Les structures conditionnelles

L'instruction if-elif-else

L'instruction if permet d'exécuter un bloc de code uniquement si une condition est vraie. Elle peut être complétée par des clauses elif (else if) et else.

# Structure if simple
age = 17
if age >= 18:
    print("Vous êtes majeur.")
else:
    print("Vous êtes mineur.")

# Structure if-elif-else
note = 15
if note >= 16:
    print("Très bien !")
elif note >= 14:
    print("Bien !")
elif note >= 12:
    print("Assez bien !")
elif note >= 10:
    print("Passable !")
else:
    print("Insuffisant !")

Indentation en Python

En Python, l'indentation (les espaces au début de chaque ligne) est significative. Elle définit les blocs de code. Tous les blocs conditionnels doivent être indentés de la même manière, généralement avec 4 espaces.

Conditions composées

Les conditions peuvent être combinées à l'aide des opérateurs logiques and (ET), or (OU) et not (NON).

# Conditions composées
age = 25
statut = "étudiant"

if age < 26 and statut == "étudiant":
    print("Vous bénéficiez du tarif jeune.")

if age < 18 or statut == "étudiant":
    print("Vous bénéficiez d'une réduction.")

if not (age >= 18):
    print("Vous êtes mineur.")

Expressions conditionnelles

Python permet d'écrire des expressions conditionnelles concises, aussi appelées opérateurs ternaires.

# Expression conditionnelle (opérateur ternaire)
age = 20
statut = "majeur" if age >= 18 else "mineur"
print(f"Vous êtes {statut}.")

# Équivalent à :
# if age >= 18:
#     statut = "majeur"
# else:
#     statut = "mineur"

Les structures itératives (boucles)

La boucle for

La boucle for permet d'itérer sur les éléments d'une séquence (comme une liste, une chaîne de caractères, ou une plage de nombres).

# Boucle for avec range
for i in range(5):  # i prend les valeurs 0, 1, 2, 3, 4
    print(i)

# Boucle for avec range et paramètres
for i in range(2, 10, 2):  # i prend les valeurs 2, 4, 6, 8
    print(i)

# Boucle for sur une chaîne de caractères
for lettre in "Python":
    print(lettre)

# Boucle for sur une liste
fruits = ["pomme", "banane", "orange"]
for fruit in fruits:
    print(fruit)

La boucle while

La boucle while exécute un bloc de code tant qu'une condition est vraie.

# Boucle while simple
compteur = 0
while compteur < 5:
    print(compteur)
    compteur += 1  # Équivalent à compteur = compteur + 1

# Boucle while avec break
while True:  # Boucle infinie
    reponse = input("Voulez-vous continuer ? (o/n) ")
    if reponse.lower() == "n":
        break  # Sort de la boucle
    print("Continuons...")

print("Fin de la boucle.")

Les instructions break et continue

Les instructions break et continue permettent de contrôler le flux d'exécution à l'intérieur des boucles.

# Utilisation de break
for i in range(10):
    if i == 5:
        break  # Sort de la boucle quand i vaut 5
    print(i)

# Utilisation de continue
for i in range(10):
    if i % 2 == 0:
        continue  # Passe à l'itération suivante si i est pair
    print(i)  # N'affiche que les nombres impairs

La clause else dans les boucles

En Python, les boucles for et while peuvent avoir une clause else qui s'exécute lorsque la boucle se termine normalement (sans rencontrer de break).

# Boucle for avec else
for i in range(5):
    print(i)
else:
    print("Boucle terminée normalement.")

# Boucle for avec break et else
for i in range(5):
    if i == 3:
        break
    print(i)
else:
    print("Cette ligne ne sera pas exécutée car la boucle a été interrompue.")

Exercices pratiques

Exercice 1 : Table de multiplication

Écrivez un programme qui affiche la table de multiplication d'un nombre saisi par l'utilisateur.

# Table de multiplication
nombre = int(input("Entrez un nombre : "))

print(f"Table de multiplication de {nombre} :")
for i in range(1, 11):
    resultat = nombre * i
    print(f"{nombre} × {i} = {resultat}")

Exercice 2 : Nombre premier

Écrivez un programme qui détermine si un nombre saisi par l'utilisateur est premier ou non.

# Vérification de nombre premier
nombre = int(input("Entrez un nombre entier positif : "))

if nombre <= 1:
    est_premier = False
else:
    est_premier = True
    for i in range(2, int(nombre**0.5) + 1):
        if nombre % i == 0:
            est_premier = False
            break

if est_premier:
    print(f"{nombre} est un nombre premier.")
else:
    print(f"{nombre} n'est pas un nombre premier.")

Exercice 3 : Jeu de devinette

Écrivez un programme qui génère un nombre aléatoire entre 1 et 100, puis demande à l'utilisateur de le deviner. Le programme doit indiquer si le nombre proposé est trop grand, trop petit ou correct.

# Jeu de devinette
import random

nombre_secret = random.randint(1, 100)
nombre_essais = 0
trouve = False

print("J'ai choisi un nombre entre 1 et 100. À vous de le deviner !")

while not trouve:
    essai = int(input("Votre proposition : "))
    nombre_essais += 1
    
    if essai < nombre_secret:
        print("C'est plus !")
    elif essai > nombre_secret:
        print("C'est moins !")
    else:
        trouve = True
        print(f"Bravo ! Vous avez trouvé en {nombre_essais} essais.")
