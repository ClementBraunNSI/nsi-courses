Exercices sur les fonctions

Numérique et Sciences Informatiques - Première

Objectifs des exercices

Cette fiche d'exercices vous permettra de pratiquer la création et l'utilisation de fonctions en Python. Vous apprendrez à définir des fonctions simples, à utiliser des paramètres, à retourner des valeurs et à documenter vos fonctions.

Les exercices sont classés par niveau de difficulté croissant. N'hésitez pas à consulter les cours sur les fonctions pour vous aider.

Exercice 1 : Fonctions simples

Écrivez les fonctions suivantes :

1.1 Fonction de salutation

Créez une fonction dire_bonjour qui affiche "Bonjour !" lorsqu'elle est appelée.

def dire_bonjour():
    print("Bonjour !")

# Test de la fonction
dire_bonjour()

1.2 Fonction avec paramètre

Créez une fonction saluer qui prend un nom en paramètre et affiche "Bonjour, [nom] !".

def saluer(nom):
    print(f"Bonjour, {nom} !")

# Test de la fonction
saluer("Alice")
saluer("Bob")

1.3 Fonction avec valeur de retour

Créez une fonction carre qui prend un nombre en paramètre et retourne son carré.

def carre(nombre):
    return nombre ** 2

# Test de la fonction
resultat = carre(5)
print(f"Le carré de 5 est {resultat}")

Exercice 2 : Fonctions mathématiques

2.1 Calcul de l'aire d'un cercle

Écrivez une fonction aire_cercle qui calcule et retourne l'aire d'un cercle à partir de son rayon.

Rappel : l'aire d'un cercle est donnée par la formule π × r².

import math

def aire_cercle(rayon):
    return math.pi * rayon ** 2

# Test de la fonction
rayon = 3
aire = aire_cercle(rayon)
print(f"L'aire d'un cercle de rayon {rayon} est {aire:.2f}")

2.2 Conversion de température

Écrivez une fonction celsius_vers_fahrenheit qui convertit une température de Celsius en Fahrenheit.

Rappel : la formule de conversion est F = C × 9/5 + 32.

def celsius_vers_fahrenheit(celsius):
    return celsius * 9/5 + 32

# Test de la fonction
temp_c = 25
temp_f = celsius_vers_fahrenheit(temp_c)
print(f"{temp_c}°C équivaut à {temp_f}°F")

2.3 Calcul de la moyenne

Écrivez une fonction moyenne qui calcule la moyenne d'une liste de nombres.

def moyenne(nombres):
    if len(nombres) == 0:
        return 0
    return sum(nombres) / len(nombres)

# Test de la fonction
notes = [12, 15, 18, 10, 14]
moy = moyenne(notes)
print(f"La moyenne des notes est {moy}")

Exercice 3 : Fonctions avec documentation

3.1 Documentation de fonction

Réécrivez la fonction aire_rectangle en ajoutant une docstring qui explique ce que fait la fonction, ses paramètres et sa valeur de retour.

def aire_rectangle(longueur, largeur):
    """Calcule l'aire d'un rectangle.
    
    Args:
        longueur (float): La longueur du rectangle.
        largeur (float): La largeur du rectangle.
        
    Returns:
        float: L'aire du rectangle (longueur × largeur).
    """
    return longueur * largeur

# Test de la fonction
print(aire_rectangle(5, 3))

# Affichage de la documentation
help(aire_rectangle)

3.2 Fonction avec annotations de type

Réécrivez la fonction celsius_vers_fahrenheit en ajoutant des annotations de type pour les paramètres et la valeur de retour.

def celsius_vers_fahrenheit(celsius: float) -> float:
    """Convertit une température de Celsius en Fahrenheit.
    
    Args:
        celsius (float): La température en degrés Celsius.
        
    Returns:
        float: La température convertie en degrés Fahrenheit.
    """
    return celsius * 9/5 + 32

# Test de la fonction
print(celsius_vers_fahrenheit(25))

Exercice 4 : Fonctions avancées

4.1 Fonction avec paramètres par défaut

Écrivez une fonction salutation qui prend un nom et un moment de la journée (matin, après-midi, soir) en paramètres, avec "jour" comme valeur par défaut pour le moment.

def salutation(nom, moment="jour"):
    """Génère une salutation adaptée au moment de la journée.
    
    Args:
        nom (str): Le nom de la personne à saluer.
        moment (str, optional): Le moment de la journée (matin, après-midi, soir ou jour).
            Par défaut "jour".
            
    Returns:
        str: La salutation complète.
    """
    if moment == "matin":
        return f"Bonjour {nom}, belle matinée n'est-ce pas ?"
    elif moment == "après-midi":
        return f"Bon après-midi {nom} !"
    elif moment == "soir":
        return f"Bonsoir {nom}, passez une bonne soirée !"
    else:  # jour
        return f"Bonjour {nom}, bonne journée !"

# Tests de la fonction
print(salutation("Alice"))
print(salutation("Bob", "matin"))
print(salutation("Charlie", "soir"))

4.2 Fonction récursive

Écrivez une fonction récursive factorielle qui calcule la factorielle d'un nombre entier positif.

Rappel : la factorielle de n (notée n!) est le produit des entiers positifs inférieurs ou égaux à n.

def factorielle(n):
    """Calcule la factorielle d'un nombre entier positif de façon récursive.
    
    Args:
        n (int): Le nombre dont on veut calculer la factorielle.
        
    Returns:
        int: La factorielle de n.
    """
    if n == 0 or n == 1:
        return 1
    else:
        return n * factorielle(n - 1)

# Tests de la fonction
for i in range(6):
    print(f"{i}! = {factorielle(i)}")

Exercice 5 : Projet - Calculatrice de fonctions

Créez un programme qui simule une calculatrice simple en utilisant des fonctions pour chaque opération.

def addition(a, b):
    """Additionne deux nombres."""
    return a + b

def soustraction(a, b):
    """Soustrait b de a."""
    return a - b

def multiplication(a, b):
    """Multiplie deux nombres."""
    return a * b

def division(a, b):
    """Divise a par b."""
    if b == 0:
        return "Erreur: Division par zéro"
    return a / b

def calculatrice():
    """Fonction principale qui simule une calculatrice."""
    print("Calculatrice Python")
    print("Opérations disponibles: +, -, *, /")
    print("Tapez 'q' pour quitter")
    
    while True:
        # Saisie de l'utilisateur
        expression = input("\nEntrez une expression (ex: 5 + 3): ")
        
        # Vérifier si l'utilisateur veut quitter
        if expression.lower() == 'q':
            print("Au revoir !")
            break
        
        # Analyser l'expression
        try:
            # Diviser l'expression en parties
            parts = expression.split()
            if len(parts) != 3:
                print("Format incorrect. Utilisez: nombre opérateur nombre")
                continue
                
            a = float(parts[0])
            op = parts[1]
            b = float(parts[2])
            
            # Effectuer l'opération
            if op == '+':
                resultat = addition(a, b)
            elif op == '-':
                resultat = soustraction(a, b)
            elif op == '*':
                resultat = multiplication(a, b)
            elif op == '/':
                resultat = division(a, b)
            else:
                print(f"Opérateur {op} non reconnu")
                continue
                
            print(f"Résultat: {resultat}")
            
        except ValueError:
            print("Erreur: veuillez entrer des nombres valides")
        except Exception as e:
            print(f"Erreur: {e}")

# Lancer la calculatrice
if __name__ == "__main__":
    calculatrice()
