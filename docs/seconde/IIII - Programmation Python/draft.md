# Exo du lundi

Écrire une fonction `indice_minimum` qui prend en paramètre une liste l et un  indice de départ i et renvoie l'indice de la valeur minimale dans la tranche allant de i à la fin dans la liste l

```python
    l = [4,2,6,5,8,9]
    >>> indice_minimum(l, 2)
        3
    >>> indice_minimum(l,1)
        1

```
